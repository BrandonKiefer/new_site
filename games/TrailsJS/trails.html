Game description:

Based on the classic game that came with Windows 3.x operating systems, this is a recreation of Trails-- now playable in the browser using JavaScript, HTML, and CSS. The controls should feel similar to classic Snake but the game is a competetion for space against a computer-controlled opponent. Read more about Trails here.

Use your keyboard's arrow keys to control your snake. The first player to run into the other player's trail, the border of the game, or his own trail is the loser-- to win you must survive!
