Projects


MyBikeDatabase is a full-stack MERN application allows users to keep track of the metrics for their bicycles' components using a very thorough and detailed bike schema. It can be used for personal record-keeping to ensure compatibility for replacing and ordering new parts while it can also has commercial potential for client/inventory management. The splash page allows users to browse through a small collection of bicycle frames and customize them with the color of their choice. When viewing a specific bike that has been submitted, the image and color of that bicycle frame will be rendered depending on its saved values. Like any full-CRUD application, submitted bicycles can be edited and deleted.

Currently hosted on Heroku, updates and plans for future deployment include a Registration and Login component for users.

Hell's Spoon is a full-stack social media and recipe-sharing application built using MongoDB, Express, EJS, and Node. Inspired from the title of one of Chef Gordon Ramsay's most popular television shows, users are able to register their own accounts to post and share their favorite recipes. Thanks to a few pieces of middleware that have been integrated into this application, registered users can also upload photos of their own favorite homemade dishes!

A play on the word RESTful API, this React application serves information about televised professional wrestling events in history. The backend is powered using Python and Django while the front end uses React to display an API that details events by promotion (wrestling organization), event year, # of matches, average match rating (based on original records from the Wrestling Observer Newsletter), best match (either based on the Wrestling Observer or my own opinion), original duration of the show, and any remarkable/historically significant notes from the event.
